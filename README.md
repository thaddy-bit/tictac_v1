# TICTAC — Trouvez vos médicaments au Congo

Solution moderne de recherche pharmaceutique et de gestion de stocks pour le Congo.

## 🚀 Fonctionnalités
- Recherche multi-médicaments en temps réel.
- Géolocalisation des pharmacies avec Leaflet.
- Paiement sécurisé des frais de recherche (MTN, Airtel, Portefeuille).
- Dashboard d'administration complet.
